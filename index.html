Oct 17, 2020 in 알고리즘, 20.10.17

문제


접근 방법

입력 할 총 화폐단위를 갯수를 입력받고, 그 다음 입력한 갯수만큼 화폐단위를 지정한다.

단위가 큰 화폐부터 / 와 % 기호를 사용하여 최소 갯수를 찾아간다.


입력시 단위는 작은 순으로 입력하기 때문에, 반복문을 실행시 끝 번호의 인덱스부터 시작하도록 설계한다.


코드



요약

  • 자바의 / (나눗셈) 과 % (나머지) 를 활용하는 문제
  • 기본적인 사고력, 수리력, 창의력을 요구하는 문제
  • 반복문을 역순으로 출력하는 문제

문제


접근 방법

이미 문제에서 5개의 문자열과 15개 이하의 길이를 정해주었다.

나중에 하나하나 문자열을 가져오기 위해서 char를 2차원 배열로 미리 생성해둔다

미리 저장된 char에서 가로방향이 아닌 세로 방향으로 반복문을 실행 함으로써, str라는 String에 추가 시키고, 혹시 빈 문자열 (== 0)이 있다면 그냥 건너뛰고 실행하도록 설계해준다!

요약

  • 2차원 배열에 대한 이해가 필요한 문제
