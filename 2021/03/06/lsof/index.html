lsof

Mar 06, 2021 in 이론노트, 기본

lsof 명령어 정리

lsof -u {name}

  • u 옵션으로 사용자를 지정 할 수 있다.

lsof /var/log/httpd/access_log

  • 특정 디렉터리내 열린 파일 보기

lsof -i {PortName}

  • 특정 포트를 사용하는 프로세스 정보 보기

lsof -i {PortName} ~ {PortName}

  • 특정 포트 범위 까지 보기

lsof -i {PortName} -P | grep java | grep LISTEN

  • grep 기능을 사용 할 수 있다

lsof -i 4 or 6

  • -i 옵션 뒤에 4(IPv4), 또는 6(IPV6) 를 지정하여 특정 IP 의 버전만 확인할 수 있다.ㅡ

  • 프로세스 중지

lsof -i

  • 모든 네트워크 포트를 표시한다.

lsof -u^lesstif -u^root

  • 사용자 지정 옵션인 -u 에 ^를 추가하면 특정 사용자는 제외할 수 있다. 다음은 lesstif 와 root 사용자가 열은 파일은 제외하고 정보를 출력한다

lsof -p {PortName}

  • 특정 프로세스가 오픈한 파일만 표시한다. 옵션뒤에 PID를 주면 된다

lsof -t -u lesstif

  • lesstif 사용자의 모든 프로세스를 중지하게 된다.

kill -KILL ‘’

  • 프로세스 중지
